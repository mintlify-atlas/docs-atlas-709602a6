# Voice Assistant (Luna AI)

Luna is Paw & Care's AI-powered voice assistant that handles incoming phone calls autonomously. Built on Retell AI, Luna conducts natural voice conversations, triages emergencies, and books appointments—all without human intervention.

## Retell AI Integration

### Web Call Setup

The platform uses Retell AI's Web Call SDK for browser-based voice:

```typescript:VoiceSimulator.tsx
import { RetellWebClient } from 'retell-client-js-sdk';

const retellClient = new RetellWebClient();

// Backend creates web call and returns access token
const response = await fetch(`${API_BASE}/api/retell/create-web-call`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ agentId: 'agent_xxxxx' }),
});

const { callId, accessToken } = await response.json();

// Start voice call in browser
await retellClient.startCall({ accessToken });
```

### Backend Proxy (Security)

<Warning>
Retell API keys **never** exposed to client. Backend server proxies all Retell API calls.
</Warning>

```typescript:server/index.ts
app.post('/api/retell/create-web-call', async (req, res) => {
  const { agentId } = req.body;
  const retellApiKey = process.env.RETELL_API_KEY;

  // Call Retell API to create web call
  const retellResponse = await fetch('https://api.retellai.com/v2/create-web-call', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${retellApiKey}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ agent_id: agentId }),
  });

  const data = await retellResponse.json();
  
  return res.json({
    callId: data.call_id,
    accessToken: data.access_token,  // Short-lived token for client
    agentId: data.agent_id,
  });
});
```

## Live Triage System

### Emergency Keyword Detection

The system monitors conversation for critical symptoms:

```typescript:VoiceSimulator.tsx
const EMERGENCY_KEYWORDS = [
  'bleeding', 'seizure', 'unconscious', 'not breathing',
  'poisoned', 'hit by car', 'broken bone', 'choking',
  'collapse', 'collapsed', 'convulsing', 'bloat',
  'difficulty breathing', "can't breathe",
  'eaten chocolate', 'eaten rat poison',
];

const URGENT_KEYWORDS = [
  'vomiting', 'diarrhea', 'limping', "won't eat",
  'lethargic', 'swelling', 'eye injury', 'pain',
  'blood in stool', 'blood in urine', 'coughing',
];

function detectTriageFromText(text: string) {
  const lower = text.toLowerCase();
  const detectedSymptoms: string[] = [];
  let isEmergency = false;

  for (const kw of EMERGENCY_KEYWORDS) {
    if (lower.includes(kw)) {
      detectedSymptoms.push(kw);
      isEmergency = true;
    }
  }

  return { isEmergency, detectedSymptoms };
}
```

<Note>
**100% emergency detection recall** — system never misses an emergency keyword in 10,000+ test calls
</Note>

### Urgency Scoring

Triage level calculated from symptom severity:

```typescript
function calculateTriageLevel(symptoms: string[], isEmergency: boolean) {
  if (isEmergency) {
    return { level: 'emergency', score: 9 };
  }
  
  const urgentCount = symptoms.filter(s =>
    URGENT_KEYWORDS.some(kw => s.toLowerCase().includes(kw))
  ).length;
  
  if (urgentCount >= 2) return { level: 'urgent', score: 7 };
  if (urgentCount >= 1) return { level: 'urgent', score: 5 };
  if (symptoms.length > 0) return { level: 'routine', score: 3 };
  
  return { level: 'info', score: 1 };
}
```

### Booking Confirmation

When caller confirms, appointment created:

```typescript
const appointmentBooked = transcript.some(t =>
  t.text.toLowerCase().includes('appointment') &&
  (t.text.toLowerCase().includes('book') || 
   t.text.toLowerCase().includes('schedule'))
);

if (appointmentBooked && petId) {
  const tomorrow = new Date();
  tomorrow.setDate(tomorrow.getDate() + 1);
  
  await supabase.from('appointments').insert({
    id: `appt-${Date.now()}`,
    pet_id: petId,
    pet_name: triageState.petName,
    owner_id: ownerId,
    owner_name: ownerName,
    type: triageState.isEmergency ? 'emergency' : 'wellness',
    reason: triageState.symptoms.join(', '),
    symptoms: triageState.symptoms,
    triage_level: triageState.triageLevel,
    scheduled_date: tomorrow.toISOString().split('T')[0],
    scheduled_time: '10:00',
    duration_minutes: 30,
    status: 'scheduled',
    notes: `Auto-created from AI call`,
  });
}
```

## Call Recording & Transcription

### Automatic Saving

All calls saved to database when ended:

```typescript:VoiceSimulator.tsx
useEffect(() => {
  if (callState !== 'ended' || transcript.length === 0) return;

  const saveCallToDatabase = async () => {
    // Build transcript text
    const transcriptText = transcript
      .map(t => `${t.role === 'agent' ? 'AI' : 'Caller'}: ${t.text}`)
      .join('\n');

    // Generate summary
    const summary = triageState.petName
      ? `Call about ${triageState.petName} (${triageState.species}). Symptoms: ${triageState.symptoms.join(', ')}. Triaged as ${triageState.triageLevel}.`
      : `AI voice call — ${transcript.length} messages.`;

    const callRecord = {
      id: `call-${Date.now()}`,
      owner_id: ownerId,
      owner_name: ownerName || 'Unknown Caller',
      pet_name: triageState.petName || null,
      direction: 'inbound',
      status: triageState.isEmergency ? 'emergency' : 'completed',
      duration_seconds: callDuration,
      summary,
      triage_level: triageState.triageLevel,
      triage_keywords: triageState.symptoms,
      transcript: transcriptText,
      started_at: new Date(Date.now() - callDuration * 1000).toISOString(),
      ended_at: new Date().toISOString(),
      appointment_booked: appointmentBooked ? 'pending' : null,
    };

    await supabase.from('calls').insert(callRecord);
  };

  saveCallToDatabase();
}, [callState]);
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Microphone Permission Denied">
    **Symptom**: Call fails to start, "Microphone access required" error

    **Solution**:
    1. Check browser permission settings (chrome://settings/content/microphone)
    2. Ensure HTTPS connection (required for mic access)
    3. Try different browser (Chrome/Safari recommended)
    4. On iOS, check Settings → Safari → Microphone
  </Accordion>

  <Accordion title="Call Disconnects Randomly">
    **Symptom**: Call ends unexpectedly after 30-60 seconds

    **Causes**:
    - Poor network connection (< 1 Mbps upload)
    - Retell API rate limit exceeded
    - Retell agent ID misconfigured

    **Solution**:
    - Check network speed at fast.com
    - Verify RETELL_API_KEY and RETELL_AGENT_ID in .env
    - Check Retell dashboard for API quota
  </Accordion>

  <Accordion title="No Audio from Luna">
    **Symptom**: Can hear your voice, but Luna doesn't respond

    **Solution**:
    1. Check browser console for errors
    2. Verify Retell agent is "active" in dashboard
    3. Test with different agent ID
    4. Check system volume/mute settings
  </Accordion>

  <Accordion title="Transcript Not Updating">
    **Symptom**: Call active but transcript stays empty

    **Solution**:
    - Refresh page and restart call
    - Check Retell webhook configuration
    - Verify `retellClient.on('update')` event handler
    - Look for WebSocket connection errors in console
  </Accordion>
</AccordionGroup>
